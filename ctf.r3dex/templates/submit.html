Submit a Contribution

Want to contribute a machine or challenge to R3dex CTF? Send us your submission via email with the required details listed below. We'll review it and add it to the platform.

Send submissions to
redexvulnos@gmail.com
Required Information
  • Machine Name - A unique name for your machine (e.g. Spectre, NightOwl)
  • Description - A brief overview of the machine's theme and attack vectors
  • Difficulty - Easy, Medium, Hard, or Insane
  • Operating System - Linux or Windows
  • User Account - Username and password for the low-privilege user
  • user.txt - The user flag
  • root.txt - The root/admin flag
  • Download Link - A link to the VM file (OVA, VMDK, etc.)
  • Writeup Link - A link to the official writeup/walkthrough
  • Creator Info - Your name/handle and a public profile link (GitHub, LinkedIn, etc.)
All submissions are reviewed before being published. Make sure your machine is fully functional and the writeup is complete.
